# Raw market data (not in git — download separately for backtests)

Place Dukascopy 1-minute bid/ask CSVs here, e.g.:

- `USA500IDXUSD_1 Min_Ask_2016.01.01_2026.07.03.csv`
- `USA500IDXUSD_1 Min_Bid_2016.01.01_2026.07.03.csv`

Alpaca paper trading does **not** need these files — it fetches SPY bars from the API.
